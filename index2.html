les catégories Mangas


Shõnen

En Occident, le mot shōnen est utilisé pour désigner un type de manga, le shōnen, qui est un type de manga dont la cible éditoriale est avant tout constituée de jeunes adolescents de sexe masculin.

Shōjo

Le shōjo manga est une bande dessinée japonaise publiée dans un magazine de prépublication dont la cible éditoriale est avant tout constituée de jeunes adolescentes.

Josei

Le josei manga est un style de manga dont la cible éditoriale est avant tout constituée par les jeunes adultes
